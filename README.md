### 开发环境配置方法

最开始，要创建一个conda环境，之后的开发都在这个虚拟环境中进行：
`conda create -n django python==3.7.*`

安装好conda环境后，每次使用要预先激活：
`conda activate django`

安装好虚拟环境后，要安装依赖包：
`pip install -r requirements.txt`

之后就可以正常开发了
